### 笔记17: 设计技巧
**标题**: 5个让PPT瞬间高大上的小技巧,设计师都在用

**封面**: PPT对比图

**正文**:
做了1年AI PPT,总结5个超实用的小技巧~
学会后,你的PPT会瞬间高大上!

✨ 技巧1: 统一字体
- 全篇最多用2-3种字体
- 标题用粗体,正文用常规
- 推荐:思源黑体 + 阿里巴巴普惠体

✨ 技巧2: 配色克制
- 主色 + 辅色 + 点缀色
- 不要超过3种颜色
- 推荐:深蓝+浅灰+橙色

✨ 技巧3: 留白
- 不要塞满内容
- 每页只讲一个重点
- 关键信息要大

✨ 技巧4: 数据可视化
- 用图表代替数字
- 柱状图对比
- 饼图占比
- 折线图趋势

✨ 技巧5: 配图选择
- 高清专业
- 风格统一
- 避免无关图片
- 推荐图库:Pexels/Pixabay

这5个技巧用上,你的PPT会专业很多! ✨
